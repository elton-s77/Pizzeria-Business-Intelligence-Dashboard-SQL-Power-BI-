# Pizzeria-Business-Intelligence-Dashboard-SQL-Power-BI

Developed an end-to-end Business Intelligence solution for a pizzeria using SQL and Power BI.

The project involved data extraction from CSV files, transformation and modeling in SQL, and the creation of interactive dashboards in Power BI using DAX measures.

The dashboard provides insights into sales performance, profit margins, product trends, and customer behavior, enabling data-driven decision-making.

Key features include KPI tracking, year-over-year comparisons, top/bottom product analysis, and dynamic filtering for deeper exploration.

🔧 Tools:
SQL
Power BI
DAX
Data Modeling
